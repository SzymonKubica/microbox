
## :gear: Required Hardware

- Arduino UNO R4 (WiFi or Minima)
- This exact [LCD display](https://www.waveshare.com/1.69inch-lcd-module.htm)
- Arduino [input shield](https://thepihut.com/products/input-shield-for-arduino?variant=27740616849)
- Any USB-C power bank ([this one](https://www.amazon.co.uk/dp/B0BCJX15NP?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1) is compatible with the battery pack)
- Any USB-C to USB-C cable (but [this one](https://www.amazon.co.uk/dp/B0CN6CKDN5?ref=ppx_yo2ov_dt_b_fed_asin_title&th=1))
- any 3D Printer to [print the enclosure](./3d-printing.md).
- 8 (or more) 8x3mm magnets (e.g. [those ones](https://www.amazon.co.uk/dp/B0CXF3R1MD?ref=ppx_yo2ov_dt_b_fed_asin_title))


You will need to solder the LCD display wires onto the input shield.
Precise instructions as to how it is done will be added here in the future.
